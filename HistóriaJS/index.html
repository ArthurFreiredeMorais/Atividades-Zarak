História do JavaScript

1995 - Criação

Brendan Eich cria o JavaScript em 10 dias na Netscape. Inicialmente chamado de Mocha.

1996 - Microsoft e JScript

Microsoft lança o JScript, versão própria do JavaScript no Internet Explorer.

1997 - ECMAScript

JavaScript é padronizado como ECMAScript pela ECMA International.

2005 - Ajax

Popularização do uso dinâmico com Ajax, iniciando a Web 2.0.

2009 - Node.js

JavaScript agora também é usado no servidor com o Node.js.

2015 - ES6

Grande atualização da linguagem, com novos recursos como let, const e arrow functions.

2020+ - Moderno e Popular

JavaScript continua evoluindo com atualizações anuais e integração com outras tecnologias.
